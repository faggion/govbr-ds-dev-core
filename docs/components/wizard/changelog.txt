# Changelog
Todas as alterações notáveis neste projeto serão documentadas neste arquivo.

## [Released]

## [4.2.1] - 16/11/2021

### Modificado
 - Alterados os tokens de cores para novo formato.

## [4.2.0] - 28/05/2021

### Adicionado
 - Comportamento Scroll no Wizard;
 - Incluído padrão de sombra para indicar etapas ocultas no Wizard;
 - Incluído orientação a respeito das distâncias iguais entre etapas;
 - Incluído orientação a repeito da Barra de Navegação Grid de 4 colunas;
 - Incluído orientação a repeito das etapas concluídas;
 - Incluído exemplos de conteúdos na Área de Conteúdo;

### Modificado
 - Exemplos das imagens no tipo vertical: distancia igual entre os Steps
 - Botões circulares na versão mobile do Wizard;
 - Exemplos das imagens com conteúdo (de texto 'lorem ipsum' para barras cinzas)

 ### Removido
  - Comportamento Scroll dos Steps (migrado para o Componente Step);
  - Comportamento Estados (migrado para o Componente Step);
  - Comportamento Uso de ícones (migrado para o Componente Step);
  - Orientação a respeito da distância entre etapas (migrado para o Componente Step);
