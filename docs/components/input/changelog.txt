# Changelog
Todas as alterações notáveis neste projeto serão documentadas neste arquivo.

 ## [Released]

## [15.2.0] - 28/12/2021

### Adicionado
- Práticas para utilizar a ação interna.
- Práticas para uso do estado foco com outros estados.

### Modificado
- Melhorias e simplificação textual da diretriz.
- Ajustes em nomes dos elementos da anatomia.
- Simplificando e verticalizando as imagens para uso em celulares e colocando no padrão visual.

### Removido
- Alguns textos da diretriz e deixando o detalhando para o padrão Formulário.
- Removido Boas práticas relacionado a internacionalização que não condiz com componente input.


## [15.1.3] - 10/12/2021

### Corrigido
 - Tokens de espaçamentos, dimensões e tipografia.
 - Alguns ajustes simples de texto.

## [15.1.2] - 20/08/2021

### Modificado
 - Cor do estado foco nas imagens que mostram esse estado.

### Removido
 - Imagens que não agregavam valor ao documento (header-behavior-search-active-1280px.png, header-behavior-search-active-768px.png, header-behavior-search-active-320px.png)

## [15.1.1] - 06/08/2021

### Modificado
 - Cor Alarm Danger (#E60000) para #E52207 (Red Vivid 50)
 - Cor Suporte 03 (#DB4800) para #D54309 (Red Warm Vivid 50)

## [15.1.0] - 01/06/2021

### Adicionado
 - Comportamento com uso de ícones representativos à esquerda do Input;
 - Boas Práticas no Uso de Máscaras de campo
 - Boas Práticas no Uso de Autocomplete
 - Tipo Input Destaque

### Modificado
 - Distância entre o campo input e a mensagem abaixo do campo para 8px (diretriz e EA)

### Corrigido
- Corrigido o alinhamento da tabela de tokens.
- corrigidos os posicionamentos das legendas das imagens.

## [15.0.2] - 09/03/2021

### Corrigido
- Foi retirado dos tokens a palavra alarm, pois não corresponde aos tokens utilizados pelos desenvolvedores.

## [15.0.1] - 26/02/2021

### Modificado
 - Correção ortográfica.

## [15.0.0] - 27/11/2020

### Adicionado

## [14.0.1] - 15/09/2020
