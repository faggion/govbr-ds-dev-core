# Changelog
Todas as alterações notáveis neste projeto serão documentadas neste arquivo.

## [Released]

## [11.2.0] - 06/09/2021

### Adicionado
 - Utilização das informações de "código da mensagem e "data e hora" para as mensagens de erro.

### Corrigido
 - Reavaliar a usabilidade no posicionamento do message.
 - A cor alternadiva indicada para o erro devido a problemas de contraste. De "--red-vivid-20" para "--red-vivid-10".

### Modificado
 - Atualização da cor verde (#168821), com opacidade 16%, para #E3F5E1.
 - Atualização da cor amarela (#FFCD07), com opacidade 16%, para #FFF5C2.
 - Atualização da cor vermelha (#E60000), com opacidade 16%, para #FDE0DB.
 - Atualização da cor azul (#155BCB), com opacidade 16%, para #D4E5FF.

## [11.1.3] - 03/08/2021

### Modificado
 - Cor Alarm Danger (#E60000) para #E52207 (Red Vivid 50)

## [11.1.2] - 09/03/2021

### Modificado
