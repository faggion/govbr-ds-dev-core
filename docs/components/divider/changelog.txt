## [1.2.0] - 23/04/2021

### Adicionado

- Adicionada a seção Anatomia
- Inclusão de diversas espessuras.
- Inclusão do estilo tracejado.
- Novas imagens de exemplo.
- Textos alt nas imagens seguindo os padrões do Design System.
- Token de espaçamento para a versão vertical do componente.

### Modificado
- O texto introdutório foi revisado
- O termo "separador" foi evidenciado para deixar a forma de uso mais clara.
- As formas de uso foram melhor definidas para se adequar aos fundamentos do design system.
- O texto de acessibilidade foi modificado para se adequar aos fundamentos atuais do design system.
- Texto de melhores práticas foi modificado para evidenciar a utilização do componente para a separação de seções ou grupos de elementos.
