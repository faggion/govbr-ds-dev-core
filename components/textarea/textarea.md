# TextArea

O Componente de TextArea .

## Dependências

Nenhuma dependência.

## Código básico

```html
<div class="br-textarea">
</div>
```

# Anatomia do componente

Este componente é formado pelos seguintes elementos obrigatórios:

- `br-textarea`: container do componente
- `textarea`: tag html 

Exemplo de uso:

```html
<div class="br-textarea">
    <textarea> ... </textarea>
</div>
```
