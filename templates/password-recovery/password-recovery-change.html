Atenção!

Observe as regras para criação de senha abaixo:

As senhas das contas do GovBr devem ter entre 8(oito) e 12(doze) caracteres.

Elas podem incluir:

Letras maiúsculas e minúsculas. As senhas diferenciam maiúsculas de minúsculas, portanto "C" é diferente de "c";

Números;

Estes símbolos: ! " # $ % & ( ) ' + , - . / : ; < -> ? @ [ \ ] ^ [ | ] ~
